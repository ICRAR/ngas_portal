NGAST File Query Page

Query the Files table.

Queries:

    Get File Last Archived
    Query to return the last file archived in the system i.e. has the most recent ingestion_date
    Get Files Within Date Range
    Returns all files within a user specified date range, default set to last 24 hours.
    File Search Form
    Query the files table based on optional parameters (file id, disk id, ingestion date and/or file status)
    Get Total File Statistics
    Displays the total number of frames and volume data.
    File Status Check
    Query the ngas_files table for the file_status column and generate a report grouped by disk_id.
    Get Insert Statistics
    Retrieve the number of distinct frames and the corresponding amount of GB of data archived during a specified period.
    Disks with File
    Retrieves which disks are files with a certain file_id.
